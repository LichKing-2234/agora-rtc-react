# agora-rtc-react

## Install

```bash
npm install --save agora-rtc-react
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'agora-rtc-react'
import 'agora-rtc-react/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
